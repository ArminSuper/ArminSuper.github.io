用Github Pages搭建一个博客

发表于 2018-02-02 | 分类于 Blog

用Github Pages搭建一个博客

Install node.js

下载node.js

Ubuntu

1    解压　tar -xvf node-v8.9.4-linux-x64.tar.xz
2    cd nnode-v8.9.4-linux-x64/bin
3    sudo ln ./node /usr/local/bin/node
4    sudo ln ./npm /usr/local/bin/npm

或者:
1    sudo apt-get install nodejs-legacy nodejs
2    sudo apt-get install npm

Windows

下载安装即可(没亲测)

Install Hexo

$ npm install hexo-cli -g
$ hexo init blog
$ cd blog
$ npm install

配置Hexo

修改_config.yml

我做了简单的修改(如下)
title: (网站标题)
subtitle: (网站副标题)
description: (网站描述)
author: (你的名字)
language: zh-Hans
timezone: (Hexo 默认使用你电脑的时区)
修改deploy
type: git
repo: (填你的仓库）
branch: master

安装主题

这里我使用的是NexT主题

$ cd your-hexo-site
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

PS. 修改站点配置文件中的theme: next

有三种Scheme选择

Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
Mist - Muse 的紧凑版本，整洁有序的单栏外观
Pisces - 双栏 Scheme，小家碧玉似的清新

这里就不一一介绍啦(上面给出了链接)

Github Pages

黑色处填写你的Github用户名

然后在站点配置文件修改repo: (填你的仓库）

PS. 上传hexo g -d即可(配置好SSH)

可以绑定你自己的域名(无需备案)
